Linux Families and Popular Distros

Linux Distribution Overview

The Linux kernel is the core of the operating system. A full Linux distribution consists of the kernel plus a number of other software tools for file-related operations, user management, and software package management. Each of these tools provides a part of the complete system. Each tool is often its own separate project, with its own developers working to perfect that piece of the system.

The vast variety of Linux distributions are designed to cater to many different audiences and organizations according to their specific needs and tastes. However, large organizations, such as companies and governmental institutions, and other entities, tend to choose the major commercially-supported distributions from Red Hat, SUSE, and Canonical (Ubuntu).

The Three Major Linux Distribution Families

The three major Linux distribution families in use today -

The Red Hat Family

Red Hat Enterprise Linux (RHEL) heads the family that includes CentOS, CentOS Stream, Fedora and Oracle Linux.

One of the key facts about the Red Hat distribution family are:

  • Fedora serves as an upstream testing platform for RHEL.
  • CentOS is a close clone of RHEL; in fact, CentOS has been part of Red Hat since 2014.
  • It supports multiple hardware platforms.
  • It uses dnf, the RPM-based package manager (covered in detail later) to install, update, and remove packages in the system.
  • RHEL is widely used by enterprises that host their own systems.

The SUSE Family

The relationship between SUSE (SUSE Linux Enterprise Server, or SLES) and openSUSE is similar to the one described between RHEL, CentOS, and Fedora.

Some of the key facts about the SUSE family are listed below

  • SUSE Linux Enterprise Server (SLES) is upstream for openSUSE.
  • It uses the RPM-based zypper package manager (we cover it in detail later) to install, update, and remove packages in the system.
  • It includes the YaST (Yet Another Setup Tool) application for system administration purposes.
  • SLES is widely used in retail and many other sectors.

The Debian Family

The Debian distribution is upstream for several other distributions, including Ubuntu. In turn, Ubuntu is upstream for Linux Mint and several other distributions. It is commonly used on both servers and desktop computers. Debian is a pure open source community project (not owned by any corporation) and has a strong focus on stability.

Some key facts about the Debian family are listed below:

  • The Debian family is upstream for Ubuntu, and Ubuntu is upstream for Linux Mint and others.
  • It uses the DPKG-based APT package manager (using apt, apt-get, apt- cache, etc., which we cover in detail later) to install, update, and remove packages in the system.
  • Ubuntu has been widely used for cloud deployments.
  • While Ubuntu is built on top of Debian and is GNOME-based under the hood, it differs visually from the interface on standard Debian, as well as other distributions.

Summary

Internet & Networking

Linux provides a full suite of tools for accessing the internet.

  • Web Browsers: Includes popular graphical browsers like Firefox and Google Chrome, as well as lightweight, text-based browsers like lynx.
  • Email Clients: Offers powerful graphical clients such as Thunderbird and Evolution, alongside efficient text-based clients like mutt.
  • Other Tools: A wide range of other utilities are available, including Filezilla for FTP, Pidgin for instant messaging, and XChat for IRC.

Office & Productivity

For office tasks, most Linux distributions come with LibreOffice, a complete office suite that provides applications for word processing, spreadsheets, presentations, and more. It is highly compatible with Microsoft Office formats.

Multimedia

Linux supports a wide array of multimedia applications for both consumption and creation.

  • Audio Players: Popular choices include Rhythmbox, Amarok, and Audacious. For audio editing, Audacity is the industry standard.
  • Video Players: Powerful and versatile players like VLC, MPlayer, and Totem can handle almost any video format.
  • Video Editors: Professional-grade video editors like Blender and Kino are also available.

Graphics & Design

Linux is a capable platform for graphic design and image manipulation.

  • Image Editing: The GIMP (GNU Image Manipulation Program) is a feature-rich, open-source alternative to Adobe Photoshop.
  • Vector Graphics: Inkscape is a powerful tool for creating and editing vector graphics, similar to Adobe Illustrator.
  • Other Utilities: Many other tools are available, including eog for image viewing and Scribus for desktop publishing.

Software Development

Linux is a first-class citizen in the world of software development, offering entire suites of tools, including a wide variety of compilers, debuggers, and integrated development environments (IDEs).
